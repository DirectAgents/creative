8 Free jQuery Slider Plugins

Imagery is an integral part of all kinds of websites and it’s more than sure that you’ll need one or even more top class image sliders and carousels for your next project. Sliders have always been a clever way to grab users attention and convey your message immediately.

In this post, I have collated the 8 best and free jQuery slider plugins, you can use in your next awesome project! They have been carefully handpicked for you to help you save time, efforts and improve websites user experience.

Ready? Jump in!

jQuery LightSlider

JQuery lightSlider is a lightweight responsive Content slider with carousel thumbnails navigation which comes with tons of features. It is fully responsive, it supports swipe and mouseDrag, it has cross-browser compatibility, it is extremely lightweight – only 9kb (minified), and much more! Check out some examples.

Ideal Image Slider

As the plugin’s creators state, is quite simply the ideal image slider in vanilla JS. It is a minimal, yet beautiful slider without any unnecessary features. In case you need to enhance it, there are more features can be added as “extensions”. Check out a demo.

Owl Carousel

This is a touch enabled jQuery plugin to create beautiful responsive carousel sliders. The new version of the famous plugin, comes with lots of new features and a friendlier API. It is fully customizable and responsive, it comes with touch and drag support, it is compatible with all modern browsers, but it has fallbacks for older browsers as well and it already has implemented some very useful build-in plugins. Check out some demos.

nanoGALLERY

nanoGALLERY is a touch enabled and fully responsive image gallery with justified, cascading and grid layout. It supports self hosted images and pulling in Flickr, Picasa and Google+ photo albums. It is very well documented and it has a lot of options to play with! Check out a demo.

UnSlider

UnSlider, is an open-source, fluid and very flexible slider plugin. It is extremely lightweight and easy to implement  because it uses simple and clean code, without any unnecessary features. Unslider, offers cross-browser compatibility, it supports keyboard controls and it adjusts at different image heights. Check out a the demo page.

Divas Slider

 

Divas Slider, is a unique, powerful multi-purpose jQuery slider plugin. It is free but it also offers a premium license at $9. Uses CSS3 transitions with jQuery fallback for old browsers, it is fully responsive and touch enabled, It comes with 5 different skins, full screen support, and lazy image  loading for better performance. Check out a demo.

Camera

Camera slideshow is an open source project, with lots of options and callbacks, making it easy to customize almost everything! It is very well structured,  it comes with video and a beautiful full screen support. Check out a demo.

FlexSlider

The guys at WooThemes created a very clean and beautiful  slider plugin, you can download and use in your projects today. With Flex Slider you can create basic slider, slider with thumbnails, video slider, carousels and so on.Check out the demo.

I hope you found this jQuery slider plugins collection useful. Please comment below if you have or know any useful related jQuery slider plugins.
